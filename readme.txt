Simplicy Random Post is a WordPress plugin that allows you to display random items in your sidebar, you can choose the month and year of publication of your articles and the category if you want. The visitor to your blog has a button to update the articles affine random walk with ease items in your blog.

== Description ==

<strong>Simplicy Random Post</strong> The options:
 
<p>Item number posted<br />
The month of publication eg December = 12 (Set to 0 to disable this option)<br />
The year of publication such as 2011 (Set to 0 to disable this option)<br />
Category selection (left on ALL to show all categories)<br />
Max number of users has preloaded<br />
Displaying a thumbnail or not<br />
Size of the thumbnail<br />
View a excerpt or not<br />
Length of the excerpt<br />
Auto refresh or not<br />
Setting the refresh interval in seconds<br />


<br />
<strong>Languages:</strong> French / English</p>


== Installation ==

Extract the zip file and just drop the contents into the folder wp-content/plugins / directory of your WordPress installation and activate the plugin from the Plugins page.
